Using Swfmill to create a library for MTASC


Create a SWF via XML through Swfmill's "simple" mode that contains library items (such as a SWF, JPG, or PNG image). Then, attach the items to the stage with code compiled with MTASC.

Configuration:


  • MTASC required - http://www.mtasc.org/
  • Swfmill required - http://iterative.org/swfmill/

    • Compilation:

    Creating the SWF with Swfmill:


  • Open a command prompt by opening Run from the Windows Start Menu, and typing cmd.exe and hitting OK.
  • Navigate to the example folder.
  • Type in the following (Changing to the location where you put Swfmill):
  • c:\Location\Where\You\Unzipped\swfmill\swfmill.exe simple library.xml IntroToSwfmill.swf
  • "simple" specifies to use Swfmill's simple XML language.
  • The second argument is the XML file containing your library.
  • The last argument is the target SWF. It doesn't have to exist, Swfmill will create it.

    • Adding classes with MTASC:


  • Open a command prompt by opening Run from the Windows Start Menu, and typing cmd.exe and hitting OK.
  • Navigate to the example folder.
  • Type in the following (Changing to the location where you put MTASC):
  • c:\Location\Where\You\Unzipped\Mtasc\mtasc.exe -version 8 -swf IntroToSwfmill.swf -cp "C:\Program Files\Macromedia\Flash 8\en\First Run\Classes" -main IfbinLogo.as
  • -version specifies to export for Flash 8
  • -swf specifies the name the of swf file in which to inject your code.
  • -main tells the swf to start with the main function.
  • -cp allows you to set a folder as a classpath. Here we're setting Flash's main class folder.
  • Finally, specify the class with your main function inside.

    • Files:


  • IntroToSwfmill.swf
  • IfbinLogo.as
