MtascWelcome


An introduction to using the MTASC ActionScript 2 Compiler.


Configuration:


  • MTASC required - http://www.mtasc.org/

    • Compilation:


  • Open a command prompt by opening Run from the Windows Start Menu, and typing cmd.exe and hitting OK.
  • Type in the following (Changing to the locations where you keep your code):
  • c:\Location\Where\You\Unzipped\Mtasc\mtasc.exe -swf "C:\Your\Custom\Location\MtascWelcome.swf" -main -header 640:480:12:FFFFFF "C:\Your\Custom\Location\MtascWelcome.as"
  • -swf specifies the name the of swf file to create
  • -main tells the SWF to start with the main function
  • -header gives information about the swf (width:height:framesPerSecond:hexBackgroundColor)
  • Make sure you include the path to your class.

    • Files:


  • MtascWelcome.swf
  • MtascWelcome.as
  • readme.html
