Flex 2 Syndication


Uses Adobe's open source XML Syndication classes for Actionscript 3 to create a feed reading application in Flex 2.


Configuration:


  • Download Atom and RSS Library from Adobe Labs

    • Compilation:


  • Create a project called "Flex 2 Syndication" in Flex Builder
  • Add or replace main.mxml from the example files
  • OR
  • Compile the example using the Flex 2 SDK
